# Instructions Locales

- Repondre et documenter en francais sauf demande contraire.
- Ne jamais automatiser d'actions LinkedIn sensibles en masse.
- Preferer les APIs publiques, ATS directs et sources officielles aux scrapers fragiles.
- Les credentials restent dans `config/.env` ou variables d'environnement, jamais dans le code.
- Toute evolution du scoring doit garder les raisons explicites dans les exports.
- Pour valider: `python -m unittest discover -s tests`, puis un run local avec `PYTHONPATH=src`.
